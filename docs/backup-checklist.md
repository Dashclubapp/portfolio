# ✅ Checklist Sauvegarde Mensuelle — DashClub

> À copier dans votre agenda le **1er de chaque mois** (ou le dernier vendredi du mois).
> Durée : **5 à 10 minutes**

---

## 📅 Sauvegarde du mois : __________ / __________

*(ex: Avril / 2026)*

---

### ÉTAPE 1 — Télécharger le code source

- [ ] Aller sur https://github.com/nanocorp-hq/triflow
- [ ] Cliquer sur **"Code"** → **"Download ZIP"**
- [ ] Le fichier `triflow-main.zip` est téléchargé

---

### ÉTAPE 2 — Renommer le fichier

- [ ] Renommer le ZIP en : `dashclub-backup-YYYY-MM.zip`
  - Exemple : `dashclub-backup-2026-04.zip`

---

### ÉTAPE 3 — Stocker la sauvegarde

- [ ] Déposer le fichier dans **Google Drive** → dossier `DashClub — Sauvegardes`
- [ ] Vérifier que le dossier contient bien le nouveau fichier

---

### ÉTAPE 4 — Vérifier la sauvegarde

- [ ] Ouvrir le ZIP — il s'ouvre sans erreur ✓
- [ ] Vérifier la présence du dossier `app/` dans l'archive ✓
- [ ] Vérifier la présence du dossier `public/` (logos) ✓
- [ ] Taille du fichier entre 1 Mo et 50 Mo ✓

---

### ÉTAPE 5 — Nettoyage (optionnel)

- [ ] Supprimer les sauvegardes de plus de 6 mois (garder les 6 dernières)
- [ ] Vérifier que le site https://dashclub.app est toujours en ligne
