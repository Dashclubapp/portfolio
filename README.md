# DashClub

Landing page and product foundation for DashClub, a SaaS platform for sport event registration aimed at French sports clubs.

## Stack
- Next.js 14 App Router
- TypeScript
- Tailwind CSS

## Local development
```bash
npm install
npm run dev
```

## Notes
- The current increment ships the marketing landing page and project scaffold.
- Neon database wiring and waitlist capture are planned for the next task.
- Naming recommendation for a broader sport brand is documented in `docs/branding-name-recommendation.md`.
